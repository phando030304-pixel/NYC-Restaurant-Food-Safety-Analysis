# NYC Restaurant Food Safety Analysis

## Overview
This project analyzes New York City restaurant inspection data to uncover food safety risks, identify common violations, evaluate cuisine performance, and compare inspection outcomes across boroughs.

## Data Source
- NYC Department of Health and Mental Hygiene (DOHMH) Restaurant Inspection Results
- Dataset provided through Analyst Builder

## Tools
- Python (Pandas)
- Google Colab
- Tableau
- GitHub

## Key Analysis
- Inspection activity by borough
- Restaurant grade distribution
- Top 10 food safety violations
- Critical vs non-critical violations
- Cuisine performance comparison
- Geographic risk analysis
- Food safety trends over time

## Objective
The goal is to provide insights that help public health officials prioritize inspections, improve food safety education, and reduce health risks across NYC restaurants.

## Project Source
This project was completed as part of a data analytics case study from Analyst Builder.
